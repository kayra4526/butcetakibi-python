# 💰 Kişisel Bütçe ve Yatırım Takipçisi

Bu proje, kullanıcıların aylık gelirlerini, harcama kategorilerini ve harcamalarını kaydederek finansal durumlarını takip etmelerini sağlayan **veri kalıcılığına sahip (JSON tabanlı)** bir Python komut satırı (CLI) uygulamasıdır.

## 🚀 Özellikler
- **Gelir Yönetimi:** Aylık toplam geliri tanımlama ve güncelleme.
- **Dinamik Kategori Yönetimi:** İhtiyaca göre özel harcama kategorileri (Market, Eğitim, Eğlence vb.) ekleme.
- **Harcama Takibi:** Eklenen harcamaları açıklamaları ve tutarları ile kayıt altına alma.
- **Veri Kalıcılığı:** Girilen tüm veriler `butce.json` dosyasında saklanır; program kapatılsa bile veriler kaybolmaz.
- **📊 Bütçe Analizi & Yatırım Öngörüsü:** - Kategori bazlı harcama dağılımı listeleme.
  - Toplam gelir ve gider analizi.
  - Yatırıma kalan net tutarın hesaplanması ve bütçe aşım uyarıları.

## 🛠️ Kullanılan Teknolojiler
- **Python 3.x**
- **JSON Modülü** (Veri saklama ve dosya I/O işlemleri için)
- **OS Modülü** (Dosya kontrolü için)
